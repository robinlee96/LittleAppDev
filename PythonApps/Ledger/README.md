# 记账本 💰

一个简单易用的个人记账应用，基于 Python 和 Tkinter 开发。

## 功能特性

### 核心功能
- 💰 **收入记录**：记录各种收入来源，支持选择发生时间
- 💸 **支出记录**：记录各种支出，支持选择发生时间
- 🔄 **转账记录**：在不同账户之间转账，两个账户余额同时变动
- 🏦 **账户管理**：管理多种类型的账户
  - 储蓄账户
  - 网络支付（支持负余额）
  - 贷款账户
  - 公积金账户

### 高级功能
- 📝 **交易编辑**：已记录的动账支持修改
- ✏️ **账户编辑**：账户信息支持修改，余额变动自动生成校正交易
- 📊 **统计报表**：根据选择的年份和月份自动刷新统计
- 📋 **交易历史**：查看和筛选交易记录
- 🏷️ **类型管理**：管理收入和支出的分类（支持主/子类别）
- 📤 **导出Excel**：将交易记录导出为Excel文件

## 界面预览

### 标签页顺序
1. **💰 动账记录**（首页）
2. **🔄 交易历史**
3. **🏦 账户管理**
4. **📊 统计报表**
5. **🏷️ 类型管理**

## 使用方法

### 运行源代码

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python main.py
```

### 使用打包的可执行文件

直接运行 `dist/记账本.exe` 即可，无需安装 Python 环境。

## 账户类型

| 图标 | 类型 | 说明 |
|------|------|------|
| 🏦 | 储蓄账户 | 普通储蓄账户，余额不能为负 |
| 📱 | 网络支付 | 网络支付账户，支持余额为负 |
| 💳 | 贷款账户 | 贷款账户，支持余额为负 |
| 🏠 | 公积金 | 公积金账户，余额不能为负 |

## 交易类型

| 图标 | 类型 | 说明 |
|------|------|------|
| 💰 | 收入 | 记录收入，增加账户余额 |
| 💸 | 支出 | 记录支出，减少账户余额 |
| 🔄 | 转账 | 账户间转账，转出减少，转入增加 |

## 数据存储

程序使用 JSON 文件存储数据，数据保存在程序同级目录的 `data/` 文件夹中：

- `accounts.json` - 账户信息
- `categories.json` - 分类信息
- `transactions.json` - 交易记录

## 打包说明

使用 PyInstaller 打包为 Windows 可执行文件：

```bash
pyinstaller --onefile --windowed --name "记账本" main.py
```

打包后的可执行文件位于 `dist/` 目录。
