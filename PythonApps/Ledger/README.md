## 项目概述
已开发完成一个完整的记账应用，包含以下核心功能：

### 主要模块
1. 数据模型 (models.py)
   
   - Account（账户）：支持储蓄账户、网络支付账户、贷款账户、公积金账户四种类型
   - Transaction（交易记录）：支持收入、支出、转账三种类型
2. 数据存储 (storage.py)
   
   - 基于JSON文件的数据持久化
   - 账户和交易记录的增删改查功能
   - 自动保存到data目录
3. 动账管理 (ledger.py)
   
   - 收入记录：增加账户余额
   - 支出记录：减少账户余额
   - 转账：在两个账户间转移资金
   - 交易撤销功能
4. 用户界面 (ui.py)
   
   - 三个主标签页：账户管理、动账记录、交易历史
   - 账户管理：添加/删除/查看账户信息
   - 动账记录：记录收入、支出、转账
   - 交易历史：查看所有交易，支持撤销
### 使用方法
运行 python main.py 启动应用，按以下步骤使用：

1. 先在"账户管理"标签页添加账户
2. 切换到"动账记录"标签页，选择账户后进行收入、支出或转账操作
3. 在"交易历史"标签页查看所有记录，可撤销交易
所有数据会自动保存在 data/ 目录下。
